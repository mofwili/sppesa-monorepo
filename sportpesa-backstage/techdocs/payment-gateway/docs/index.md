# payment-gateway

## Overview

| Property | Value |
|---|---|
| Owner | Payments Team |
| Lifecycle | production |
| On-Call | payments-oncall@sportpesa.com |

The Payment Gateway integrates with M-Pesa, Visa, Mastercard and mobile money providers to process deposits and withdrawals for SportPesa players across East Africa.
